# Create React App example

## How to use

Download the example:

```
cd `download-folder`
```

Install it and run:

```bash
npm install
npm run start
```

## Used Node Modules

- React (obviously)
- Material UI
- Kinvey JavaScript SDK
- Prop types
- Redux
- Redux Thunk
- Connected react router
- React DOM
- Rect router
- Rect router dom
- Recompose
- History
